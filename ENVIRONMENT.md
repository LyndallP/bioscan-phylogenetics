# Bioscan Phylogenetics Environment

## Activate environment before running scripts:
```bash
conda activate bioscan-phylo
```

## Installed tools:
- mafft (alignment)
- epa-ng (phylogenetic placement)
- gappa (placement analysis)
- raxml-ng (tree building)
- vsearch (sequence clustering)
- blast (sequence search)
- samtools (sequence indexing)
- Python packages: pandas, biopython, openpyxl

## To deactivate:
```bash
conda deactivate
```
