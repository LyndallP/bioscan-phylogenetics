🔬 UK BIOSCAN Phylogenetics

Phylogenetic placement of UK BIOSCAN insect specimens onto BGE reference trees, revealing biodiversity coverage and highlighting species gaps in the UK barcode library.

About This Resource

The BIOSCAN UK project is systematically collecting and DNA barcoding arthropods across the UK, generating comprehensive COI-5P barcode coverage for British insect diversity. This visualization platform places BIOSCAN specimens onto high-quality reference phylogenies from the Bioscan Genomics Europe (BGE) project, creating interactive trees that span 450+ arthropod families across 40 orders.

Each tree integrates multiple data sources: BIOSCAN UK specimens, European reference barcodes, Darwin Tree of Life (DToL) genome assemblies, and UK Species Inventory (UKSI) taxonomy. The modular design allows new datasets—such as genome-sequenced specimens or regional barcode initiatives—to be seamlessly added to existing trees.

Why This Matters

These trees reveal the UKSI gap analysis in action: which British species lack COI barcodes, where taxonomic uncertainties exist, and which lineages would benefit from genome sequencing. By visualizing BAGS quality grades (Pentinsaari et al. 2020), BIN assignments, and phylogenetic placement confidence together, the trees support biodiversity surveillance, taxonomic validation, and genome prioritization decisions.

The UKSI gap analysis underlying these visualizations is part of the broader UKBOL (UK Barcode of Life) initiative. See the UKBOL species coverage overview for context on which British species have COI barcodes.

How to Use These Trees

  1. Click a family name in the table below to open its interactive tree in Taxonium (opens in new tab).
  2. Explore the data: In Taxonium, use the Color by dropdown to visualize different metadata dimensions—BAGS grades, BIN quality issues, placement confidence, geography, UKSI status, and more.
  3. Click any tip node to see full specimen metadata, including direct links to BOLD records, NCBI BLAST searches, GBIF distribution maps, NBN Atlas records, and DToL genome assemblies (where available).
  4. Use the Search button (top-right in Taxonium) to find specific specimens, species, BINs, or metadata values. See the searchable terms table below.
  5. Share your view: After exploring, copy the browser URL—it encodes the current zoom, coloring, and search state, creating a shareable link that works without file access.

Search Tips

  • Exact match: Search bags_grade: A for all Grade A specimens
  • Partial match: Search Empis to find all Empis species
  • BIN search: Search BOLD:ACD3004 to see all specimens in that BIN
  • Geographic filter: Search United Kingdom or British Isles
  • Quality flags: Search shares_BIN_with_other_species to find potential misidentifications
  • UKSI gaps: Search UKSI_no_specimens to find UKSI-listed species still lacking BIOSCAN coverage
